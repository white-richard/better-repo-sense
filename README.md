# Better Repo Sense

A simple TUI tool to probe GitHub contributions.

## Installation

This project uses [uv](https://github.com/astral-sh/uv) for management:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then, install the dependencies:

```bash
uv sync
```

## Usage

```bash
python main.py
```

## Inspiration

This idea is inspired by [RepoSense](https://github.com/RepoSense/reposense).
